1. Counting Pretty Numbers 
A number is considered “pretty” if its last digit is 2, 3, or 9. For each test case, we are given two integers L and R, and we must count how many numbers between L and R (inclusive) end with one of these digits.

To solve this, we simply check every number in the range from L to R and look at its last digit (which can be found using number % 10). If the last digit is 2, 3, or 9, we increase our count. Since the maximum value of R is only 
10
5
10
5
, this straightforward approach is efficient enough even for up to 100 test cases.

For example, in the range 1 to 10, the numbers ending in 2, 3, or 9 are 2, 3, and 9, so the answer is 3. In the range 11 to 33, the pretty numbers are 12, 13, 19, 22, 23, 29, 32, 33, giving a total of 8.
SOURCE CODE:
import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();
        while (T > 0) {
            int L = sc.nextInt();
            int R = sc.nextInt();
            int count = 0;
            for (int i = L; i <= R; i++) {
                int d = i % 10;
                if (d == 2) {
                    count++;
                } else if (d == 3) {
                    count++;
                } else if (d == 9) {
                    count++;
                }
            }
            System.out.println(count);
            T--;
        }
    }
}
2.Y - Birthday of Anabelle
Annabelle is 20 years old, and she has given a condition that she will accept the two gifts only if their total price equals her age multiplied by 100. Since her age is 20, the required total price is 20 × 100 = 2000. For each test case, we are given the number of items in the store and a list of their prices. Vishal must choose exactly two different items (he cannot buy the same item twice), such that their sum equals 2000.

This problem is essentially checking whether there exists a pair of distinct elements in the list whose sum is 2000. A simple and efficient way to solve this is by using a set. As we iterate through the list of prices, for each price x, we check whether 2000 − x has already been seen. If yes, then we have found two different items whose sum is 2000, and we print “Accepted”. If no such pair is found after checking all items, we print “Rejected”.

In the sample input, the list of prices is 10 2 1000 50 1000. Here, there are two different items priced at 1000 each. Their sum is 1000 + 1000 = 2000, which satisfies the condition (20 × 100). Therefore, the output is “Accepted”.
source code:
import java.util.Scanner;
public class Main{
public static void main(String[] args){
Scanner sc=new Scanner(System.in);
int T=sc.nextInt();
while(T>0){
int N=sc.nextInt();
int[] arr=new int[N];
for(int i=0;i<N;i++){
arr[i]=sc.nextInt();
}
int target=2000;
int found=0;
for(int i=0;i<N;i++){
for(int j=i+1;j<N;j++){
if(arr[i]+arr[j]==target){
found=1;
break;
}
}
if(found==1){
break;
}
}
if(found==1){
System.out.println("Accepted");
}else{
System.out.println("Rejected");
}
T--;
}
}
}
===V - Solve The Case
Since the array is already sorted, all duplicate elements will appear next to each other. So, to remove duplicates, we only need to keep the first occurrence of each element and skip the repeated ones.

For each test case, read the size n and the sorted array. Then iterate through the array and compare each element with the previous one. If it is different, include it in the new array; otherwise, skip it. Because the array is sorted, this guarantees that each element appears only once in the result.

This approach runs in O(n) time for each test case and does not require extra complex data structures.

For example:

Input: 1 1 2
The first 1 is kept, the second 1 is skipped (duplicate), and 2 is kept.
Output: 1 2

Input: 0 0 1 1 1 2 2 3 3 4
Keep the first occurrence of each number and skip the consecutive duplicates.
Output: 0 1 2 3 4

Thus, for every test case, print the modified array with elements separated by spaces, ensuring each element appears only once.
SOURCE CODE:
import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();
        for (int t = 0; t < T; t++) {
            int n = sc.nextInt();
            int[] arr = new int[n];
            for (int i = 0; i < n; i++) {
                arr[i] = sc.nextInt();
            }
            int prev = arr[0];
            System.out.print(prev);
            for (int i = 1; i < n; i++) {
                if (arr[i] != prev) {
                    System.out.print(" " + arr[i]);
                    prev = arr[i];
                }
            }
            System.out.println();
        }
    }
}
========W - Multiply the Array 
For each test case, you are given an array of N integers, and your task is to compute the product of all elements in the array.

Since the maximum value of each element is 15 and the maximum size of the array is 10, the largest possible product is:

This value fits within a 64-bit integer (long long in C++ / long in Java), so you should use an appropriate data type to avoid overflow.

Approach:

Read the number of test cases T.

For each test case:

Read N.

Read the N integers.

Initialize a variable product = 1.

Multiply each element with product.

Print the final product for each test case.

Example:

Input:

1
2
2 2

Explanation:
Product = 2 × 2 = 4

Output:

4

If any element in the array is 0, the product will immediately become 0.

This solution runs in O(N) time per test case and works efficiently within the given constraints.
SOURCE CODE:
import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();
        for (int t = 1; t <= T; t++) {
            int N = sc.nextInt();
            long product = 1; 
            int i=0;
            for ( i = 0; i < N; i++) {
                int num = sc.nextInt();
                product = product * num;
            }
            System.out.println(product);
        }
    }
}
